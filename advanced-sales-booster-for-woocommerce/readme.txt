== Description ==

**Advanced Sales Booster for Woocommerce** empowers your store with advanced features designed to enhance sales and user experience. The plugin includes the following key functionalities:

* AI-driven Frequently Bought Together Products.
* Exciting discounts based on product quantities.
* Discount tables displayed on product pages.
* Upsell and cross-sell products on the cart and checkout pages.
* Customizable sliders for upsell products.
* Advanced cart condition-based upsell displays.
* Custom email marketing templates for specific users or roles.
* Sales notifications in popups with full customization options.
* Price offer functionality for customers with automated approvals.
* Visitor count display for products.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings under WooCommerce > Settings > Advanced Sales Booster.
