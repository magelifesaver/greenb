Add delivery date and time management for WooCommerce delivery orders.

== Description ==

The **Scheduling Deliveries for Delivery Drivers** plugin enhances WooCommerce by adding the ability to manage and schedule **delivery dates and times** for orders. This plugin focuses on helping store owners and delivery drivers organize when deliveries will take place, providing an intuitive interface for setting and viewing delivery schedules.

### Key Features:
- Add and manage delivery dates and times for WooCommerce orders.
- Store delivery date and time information in the order details.
- Use jQuery UI Datepicker for easy selection of delivery dates and times.
- Drivers can view the scheduled delivery date for their assigned orders.
- Admins can modify delivery dates directly from the WooCommerce order page.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/scheduling-deliveries-for-delivery-drivers` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin via the 'Plugins' screen in WordPress.
3. Go to local delivery drivers for woocommerce settings to manage and set delivery dates and times.
 

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin adds a **delivery date and time** management feature to WooCommerce. You can set delivery dates for each order, and this information is displayed to drivers and admins alike.

= How is the delivery date added to orders? =

A delivery date field is available in the WooCommerce order details. Admins can assign a delivery date and time using a date picker integrated with jQuery UI.

= Can drivers see the delivery dates? =

Yes, drivers can view the scheduled delivery date and time for their assigned orders from their panel.
