== Description ==

Need a simple way to stay organized inside your WordPress admin area? **Sticky Notes for WP Dashboard** lets you add customizable sticky notes right to your dashboard—just like the ones on your desk, but smarter.  

Use it to jot down reminders, create to-do lists, or leave quick notes for other users. Each note can be styled, resized, and moved around to fit the way you work. Whether you're managing a personal site or running a team, Sticky Notes makes it easy to keep important information front and center.  

### Why you'll love Sticky Notes for WP Dashboard:
– Create as many sticky notes as you need, anywhere in the admin dashboard  
– Change colors, fonts, and themes to match your style  
– Drag and drop notes to position them exactly where you want  
– Show or hide notes with one click  
– Duplicate, archive, and organize notes easily  
– Control who can access notes with user role restrictions  
– Option to hide notes on specific admin pages for a cleaner view  

With an intuitive interface and zero setup required, Sticky Notes for WP Dashboard is a must-have productivity tool for any WP site owner.

== Installation ==

1. In your WordPress admin dashboard, go to **Plugins <span aria-hidden="true" class="wp-exclude-emoji">→</span> Add New**.  
2. Search for **Sticky Notes for WP Dashboard**.  
3. Click **Install Now**, then **Activate**.  

Once activated, you’ll see the **Sticky Notes** menu in the WP admin toolbar.

== Frequently Asked Questions ==

= How can I create a new sticky note? =  
Go to any page in the WordPress admin dashboard. From the top admin bar, click **Sticky Notes <span aria-hidden="true" class="wp-exclude-emoji">→</span> Create New Note**.
Note: If you enabled the setting to allow notes only on specific admin screens, this option will be available only on those screens.

= Can I hide or minimize the notes? =  
Yes. You can globally hide or show all notes using the toggle under the **Sticky Notes** menu in the admin toolbar.

= Are sticky notes resizable and movable? =  
Yes. You can drag notes by their header to reposition them and resize them as needed.

= Can I change the color and style of a note? =  
Yes. Each note has its own customization menu on the top-left corner to change themes and fonts.

= Can I disable sticky notes on specific admin screens? =  
Yes. From the plugin settings, select the admin screens where you want to hide notes. If a screen is not listed, use the `wb_stn_hide_on_these_pages` filter to hide it programmatically.
